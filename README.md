# Chai Empire

Offline idle-upgrades mobile game concept and prototype for Indian audiences.

## Core Fantasy
Start with a small chai stall. Serve customers automatically. Upgrade speed, staff, menu, quality, and reputation. Expand from a roadside stall into a full chai empire.

## Design Influences
Research direction considered:
- idle/incremental progression loops
- business sims like Lemonade Stand
- time-management progression patterns like Diner Dash

This project is intended to adapt those proven loops into an India-native chai business fantasy, not copy them.
