# Chai Empire UI + Art Pack

Production-facing documentation for the first playable visual pass of **Chai Empire** on iOS and Android.

## Files
- `VISUAL_STYLE_GUIDE.md`
- `SCENE_AND_UI_GUIDE.md`
- `ASSET_PLAN.md`
- `MONETIZATION_AD_PLACEMENT.md`

## Intended Use
Use this pack to align:
- game UI layout
- 2.5D art direction
- asset scope for v1 production
- animation priorities
- future-safe ad placement that protects retention

## Build Assumptions
- portrait mobile layout first
- Unity, React Native canvas, Godot, or similar layered 2D runtime
- faux-3D / 2.5D look created with painted depth, shadow planes, scale separation, and light animation
- optimized for mid-range Android devices first, then iOS polish parity
